# Output

Use this structure for troubleshooting responses:

1. Symptom
2. Key Evidence
3. Root Cause or Ranked Hypotheses
4. Recommended Next Step
5. Follow-Up Fix In Repo State, if applicable

## Rules

- Quote only the smallest necessary evidence.
- When confidence is partial, say what would confirm or eliminate each hypothesis.
- Rank remediation options by safety when there is more than one reasonable path.
