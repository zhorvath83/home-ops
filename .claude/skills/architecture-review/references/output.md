# Output

Use this structure for design responses from this skill:

1. Context
2. Options Considered
3. Recommended Direction
4. Why This Fits `home-ops`
5. High-Level Implementation Requirements
6. Risks And Open Questions

## Rules

- Keep the implementation section high level only.
- Do not slip into file-by-file edits.
- If one option is clearly dominant, still name the rejected alternatives briefly so the tradeoff is explicit.
